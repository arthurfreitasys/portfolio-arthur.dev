# Portfólio - Arthur Silva

Meu portfólio pessoal desenvolvido para apresentar minha trajetória, habilidades e projetos como estudante de Análise e Desenvolvimento de Sistemas.

## Sobre o projeto

Este projeto foi criado com o objetivo de funcionar como minha vitrine profissional no GitHub e na web.  
Nele, apresento um pouco sobre mim, minhas principais habilidades e alguns projetos que desenvolvi ou participei.

O site foi pensado com um design limpo, direto e profissional, buscando destacar minha identidade como desenvolvedor e facilitar a navegação para recrutadores, colegas e visitantes.

## Objetivos

- Apresentar meu perfil profissional
- Mostrar minhas habilidades técnicas
- Exibir projetos desenvolvidos
- Reforçar minha presença online como desenvolvedor

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Git
- Figma

## Estrutura do portfólio

O site está dividido em seções principais:

### Início
Apresentação inicial com identidade visual, nome e atalhos para redes e currículo.

### Sobre mim
Breve descrição sobre minha formação, perfil e objetivos na área de tecnologia.

### Habilidades
Lista de tecnologias e ferramentas que fazem parte da minha trajetória de aprendizado e desenvolvimento.

### Projetos
Espaço dedicado à apresentação de projetos, com descrição e tecnologias utilizadas.

## Projetos em destaque

### Colégio Sanquim
Sistema web para uma escola local, com foco em registro de frequência, notas de alunos e planejamento de aulas, promovendo organização e transparência entre professores, alunos e gestores.

### Portfólio Pessoal
Projeto desenvolvido para apresentar minhas habilidades, experiências e identidade visual como desenvolvedor.
